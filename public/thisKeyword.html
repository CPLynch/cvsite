How to always find what the this keyword is.

Examine the call site and apply the following rules from the spec from low to high priority:

  1. Default binding - if your in strict mode the this keyword defaults to undefined, if not in strict mode it defualts to the global variable.
  2. Implicit Binding - When a function is called via an object property reference (ie. object.property();) then that object becomes the this keyword.
  3. Explicit Binding - When using .bind() .call() or .apply() somewhere in the call stack they take an argument that is used as the this keyword.
  4. The New keyword - if the function is called with the new keyword use that object as this. The new keyword overrides hard binding.
